## 2. 色彩 Tokens（全站 4 色，0 渐变）

```css
--paper:        #FAF8F3;   /* 背景纸面，类似宣纸 */
--ink:          #1A1614;   /* 主文，略带暖意的黑 */
--muted-ink:    #6B6660;   /* 副信息、注脚 */
--vermilion:    #A23F3F;   /* 朱砂，全站使用面积 ≤ 1% */
--hairline:     #E8E2D5;   /* 极淡分隔线 */
```

**纪律**：朱砂只用在 — 印章 / 关键引文左侧 1px 竖线 / hover 下划线。**稀缺度即是力量**。

## 3. 字体系统

| 用途 | 字体 | 字重 |
|---|---|---|
| 中文显示与正文 | Noto Serif SC（思源宋体） | 200 / 300 / 400 / 500 |
| 装饰序号 01–05 | Cormorant Garamond | 300 |

**禁用**：Inter / Roboto / 思源黑体 / 任何无衬线作为标题。**全站 1 个中文字体家族**。

---

## 4. 间距、圆角、阴影

```
基本单元   8px
栏宽       720px（正文）/ 960px（双栏）/ 1200px（外框）
节奏       32 / 48 / 80 / 160（四级垂直节奏）
圆角       0
阴影       0（极少数用 0 1px 0 rgba(26,22,20,0.04) 模拟纸张）
分隔线     1px solid var(--hairline)，长度通常 ≤ 240px
```

---

## 5. 动效

```
缓动      cubic-bezier(0.2, 0.6, 0.2, 1)
入场      opacity 0→1, translate-y 12px→0, 700ms
hover     border-bottom 0→1px, 240ms（无色块变化）
reduced-motion → 全部退化为 opacity 渐显
```

**禁用**：scale 弹跳 / 3D 翻转 / 视差 / 鼠标跟随。

## 7. 文案语气（Tone of Voice）

| 不要 | 要 |
|---|---|
| 赋能 / 干货 / 认知升级 / 底层逻辑 | 一段 / 一程 / 慢慢 / 顺手 |
| 立即报名 / 限时 / VIP | 同行 / 走一程 / 写信告诉我 |
| 21 天搞定 / 5 分钟读懂 / 必读 | 慢慢读 / 读两遍三遍 / 一遍不一样 |
| 大师 / 专家 / 导师 | 工程师 / 普通人 / 同路人 |
| 感叹号、表情、emoji | 句号、省略号、留白 |

**核心句式**：短句、否定句、自嘲。**不卖力，不喊话**。
